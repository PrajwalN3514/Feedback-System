# Feedback System

This is a web-based feedback collection system developed using HTML, CSS, JavaScript, Node.js, and MongoDB. It supports user and admin roles. Users can log in and submit feedback, while admins can log in to view and export all collected feedback.

Technologies Used:
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js
- Database: MongoDB (MongoDB Compass or local instance)

How to Run the Project:
1. Ensure you have Node.js and MongoDB installed.
2. Open your terminal and navigate to the backend project folder.
3. Run `npm install` to install dependencies.
4. Start the server by running:
   node admin.js
5. Make sure MongoDB is running (`mongod` or MongoDB Compass).
6. Open a browser and go to:
   http://localhost:5000

MongoDB Setup:
- Use MongoDB Compass or CLI
- Create a database named `feedback_system`
- Collections will be created automatically when feedback is submitted
